# Island-Generator
Generates rectangular maps that look like Islands, uses SDL2.
Download .exe along with the Assets and .dlls to use.
ESC - Kill,
F1 - Night On/Off
F2 - Empty Map
F3 - Cicle through tiles
F4 - Cells shading
F5 - Switch mouse cursor highlight
F7 - Randomize every tile
F8 - Generate Map
